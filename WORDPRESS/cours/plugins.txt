[Plugins/Extensions]
	un plugin = une extension = un module = une add-on = un bundle = un composant. Tous ces termes désigne une fonctionnalité.
	
	Lorsque nous souhaitons une nouvelle fonctionnalité (obtenir un comportement particulier ou un affichage spécifique) qui n'est pas prévu par l'administration wordpress, nous allons rechercher, installer, activer, et configurer des plugins.
		
	A chaque problématique, nous allons recherche le plugin qui pourra nous rendre service !

Autres plugins pouvant être utiles :
	- PHP Code Widget - permet d'ajouter du code php exécuté dans les widgets.
	- Insert PHP - permet d'ajouter du code php exécuté dans les contenus (pages, articles, etc).	
	- Page scroll to id https://wordpress.org/plugins/page-scroll-to-id/	
	- wp no category base => permet d'éviter d'avoir écrit "category" dans l'url.

	D'une manière générale, rendez-vous dans la partie Extension > Ajouter > Populaire pour connaitre les plugins qui ont la cote !

03/01.	[Plugins/Extensions] => SIMPLE LIGHTBOX

		Problématique : Si vous afficher l'un de vos articles ou pages et qu'une photo est insérée, vous verrez que cette photo est cliquable et qu'elle emmène soit sur la page en cours, soit sur elle même. Ce qui n'est pas idéal.
		
		Extension > Ajouter > Saisir dans la case de recherche : "Simple Lightbox"
		Extension > Ajouter > Installer "Simple Lightbox" (Par Archetyped)
		Extension > Ajouter > Activer "Simple Lightbox"
		
		Apparence > Lightbox > 	Cette sous-partie n'existait pas avant l'installation du plugin. Lorsque nous ajoutons un plugin, il faut toujours estimer l'apport du plugin en vérifiant ce qui a changé en backoffice et/ou en frontoffice.
		
		FRONT >					Si vous afficher l'un de vos articles ou pages et qu'une photo est insérée, vous verrez que cette photo est cliquable et qu'elle s'ouvre dorénavant dans une fenêtre interne (appelée lightbox) !
		
		Si jamais le clic sur une image ne fait pas apparaitre la box contenant l'image zoomé (effet attendu). il faut retourner dans l'article (ou la page) contenant l'image et choisir dans le paramétrage l'option "Lier au fichier média".

		Conclusion : Nous avons installé un plugin qui a amélioré l'affichage de nos photos, lors d'un clic, sur notre site Wordpress.

03/02.	[Plugins/Extensions] => META SLIDER

		Problématique : Notre page d'accueil est plutôt "figée", pas très dynamique. Si nous souhaitons afficher un slider sur notre page d'accueil, nous ne pouvons pas le faire avec wordpress de base, nous allons donc rechercher un plugin.

		Extension > Ajouter > Saisir dans la case de recherche : "Meta Slider"
		Extension > Ajouter > Installer "Meta Slider" (Par Matcha Labs)
		Extension > Ajouter > Activer "Meta Slider"
		
		Meta Slider > Cette sous-partie n'existait pas avant l'installation du plugin. Lorsque nous ajoutons un plugin, il faut toujours estimer l'apport du plugin en vérifiant ce qui a changé en backoffice et/ou en frontoffice.
		
		Meta Slider > Une partie gratuite et une partie payante existe pour ce plugin.
		Meta Slider > (+) Create your first slideshow
		Meta Slider > Renommer : Mon Diaporama	
		Meta Slider > Ajouter une diapositive (ajouter 5 images de grande taille, orienté paysage, type bandeau). Puis cliquer sur "Ajouter au diaporama".
		Meta Slider > Sur chaque image (diapositive) nous pouvons ajouter un lien, une legende, gérer l'attribut alt, etc.
		Meta Slider > Dans la colonne de droite plusieurs réglages sont possible (largeur, hauteur, effet, vitesse/délai, etc.).
		Meta Slider > Dans la colonne de droite, un cadre nommé "Utilisation" contient le shortcode à insérer. Exemple : [metaslider id=10]
		
	[ShortCode]
		Un ShortCode est un morceau de code entre crochet, détecté et reconnu par le système wordpress pour effectuer une action ou générer un affichage.
		
		Pages > Accueil > Modifier > Modifier la page d'accueil et ajouter votre slider, pour cela 2 solutions :
			1 - Cliquer sur le bouton "Add Slider" et choisissez votre slider
			2 - copier/coller le code correspondant à votre slider, directement dans la zone de contenu, exemple : [metaslider id=10]
		
		FRONT >		 Afficher votre page d'accueil et vous pourrez constater que vous qu'un diaporama s'affiche !
		
		Conclusion : Nous avons installé un plugin qui nous permet de bénéficier d'un diaporama/slider, sur notre site Wordpress.

03/03.	[Plugins/Extensions] => CONTACT FORM 7

		Problématique : Chaque site possède un formulaire de contact et nous n'en avons pas.
		
		Extension > Ajouter > Saisir dans la case de recherche : "Contact Form 7"
		Extension > Ajouter > Installer "Contact Form 7" (Par Takayuki Miyoshi)
		Extension > Ajouter > Activer "Contact Form 7"
		
		Contact > Cette sous-partie n'existait pas avant l'installation du plugin. Lorsque nous ajoutons un plugin, il faut toujours estimer l'apport du plugin en vérifiant ce qui a changé en backoffice et/ou en frontoffice.
		
		Contact > Un formulaire de contact est déjà disponible : Formulaire de contact 1
		Contact > Copier le ShortCode du formulaire : [contact-form-7 id="14" title="Formulaire de contact 1"]
		
		Pages > Ajouter > Créer une page nommée "Contact".
		Pages > Ajouter > Coller le ShortCode correspondant au formulaire : [contact-form-7 id="14" title="Formulaire de contact 1"]
		Pages > Ajouter > Enregistrer votre page.
		
		Apparence > Menus > Cliquez sur les pages "Afficher tout" : choisir Contact et cliquer sur "Ajouter au menu". (si vous n'aviez pas activer l'option ""Ajouter automatiquement les pages de premier niveau à ce menu").
		
		FRONT >			Afficher la page contact, vous verrez que vous y trouverez un nouveau formulaire !
						Il est possible d'envoyer un email (si l'antivirus est desactivé et si le smtp est correctement configuré dans le php.ini de wamp).

		Contact > Créer un formulaire > Cette partie vous permet de créer d'autres formulaires.
		Contact > Intégration > Cette partie permet de mettre en place un captcha (code de vérification pour éviter le spam ou l'ajout de commentaire par des robots).
		
		Conclusion : Nous avons installé un plugin qui nous permet d'obtenir rapidement un formulaire mais aussi tout autre type de formulaire, sur notre site Wordpress.

03/06.	[Plugins/Extensions] => PAGE BUILDER PAR SITEORIGIN

		Problématique :  Nos pages web se ressemble et il peut être parfois utile de modifier l'agencement d'une page pour mettre en avant son contenu.
		
		Extension > Ajouter > Saisir dans la case de recherche : "Page Builder" 
		Extension > Ajouter > Installer "Page Builder" (Par siteorigin).
		Extension > Ajouter > Activer "Page Builder" 
		
		Réglages > Page Builder > Cette sous-partie n'existait pas avant l'installation du plugin. Lorsque nous ajoutons un plugin, il faut toujours estimer l'apport du plugin en vérifiant ce qui a changé en backoffice et/ou en frontoffice.
		
		Pages > Ajouter > Créer une page nommée "builder".
		Pages > Ajouter > Cliquez sur l'onglet "Page Builder".
			- Ajouter un widget => permet d'afficher un widget directement dans un contenu.
			- Ajouter une rangée => permet de créer des zones sur sa page web.
			- Prédéfinie => permet d'insérer un agencement prédéfinie sur sa page web.
				- Il est probable qu'on vous adresse un message : " Pack de widgets SiteOrigin pour utiliser le widget SiteOrigin_Widget_Features_Widget. "
				- Wordpress vous propose un lien permettant d'installer le plugin nommé "SiteOrigin Widgets Bundle" afin de profiter des mises en forme prédéfinie.
		Pages > Ajouter > Enregistrer votre page.
		
		Si on ne vous a pas proposé d'installer le plugin "SiteOrigin Widgets Bundle" ou que la mise en page n'est pas concluante.
		Extension > Ajouter > Saisir dans la case de recherche : "SiteOrigin Widgets Bundle" 
		Extension > Ajouter > Installer "SiteOrigin Widgets Bundle" (Par siteorigin).
		Extension > Ajouter > Activer "SiteOrigin Widgets Bundle"
		Extension > SiteOrigin Widgets Bundle > Permet d'activer / désactiver certains composants d'affichage.
		
		Pages > Ajouter > Modifier la page nommée "builder".
		Pages > Ajouter > Cliquez sur l'onglet "Page Builder".
			- Prédéfinie => Vous devriez pouvoir vous servir de tous les templates (agencements) prédéfinis.
			
			Choisir les styles prédéfinie, installer les plugins dépendant si nécessaire. Remplacer les textes et les images dans les emplacements prédéfinies.
			
			Choisir rangée. prendre 2 ou 3 rangée et le répéter l'opération sur 5 lignes pour ensuite les remplir avec les widgets texte, widget image et widget block wysiwyg.
			
			Choisir Widget pour effectuer une démo.
			
		Pages > Ajouter > Enregistrer votre page.
					
		Apparence > Menus > Cliquez sur les pages "Afficher tout" : choisir builder et cliquer sur "Ajouter au menu". (si vous n'aviez pas activer l'option ""Ajouter automatiquement les pages de premier niveau à ce menu").
			
		FRONT >		Afficher la page builder, vous verrez que vous y trouverez un agencement selon votre choix.
		
		Apparence > Widgets > Ce plugin fournit de nombreux widgets intéressants !
		
		Conclusion : Nous avons installé un plugin qui nous permet de nous assister dans la création d'affichages personnalisés, sur notre site Wordpress.
		
		A noter : Le plugin payant Visual Composer (voir dev-plugins/js_composer.zip) permet également d'effectuer des mises en forme avancée.
